# Marketing-Campaign-Effectiveness
Exploratory Data Analysis Project: Marketing Campaign Effectiveness Analysis is a Python-based data analytics project that evaluates marketing campaign performance using data-driven insights and an AI-powered interactive dashboard. The project involves data cleaning, ETL processing, exploratory data analysis, and visualization to understand trends in ROI, conversion rates, and marketing channel performance. Using libraries such as Pandas, Matplotlib, Seaborn, and Sweetviz, the analysis explores univariate, bivariate, and multivariate relationships to uncover meaningful patterns. The cleaned dataset is stored in SQLite for structured querying, and a Streamlit dashboard generated with the help of Gemini AI presents insights interactively to support smarter marketing decisions.
